# Bash-Projekte
3 kleine Projekte die im Rahmen einer Modularbeit umgesetzt wurden. 

## Email_zuweisung
Mengen von Emails werden durchiteriert und für jede einzelne werden Briefe und Kuverts erstellt. 

## Klassen_erzeugung
Erstellen einer festgelegten Menge an Dummy-Data. Die Idee dahinter ist Bereitstellung von Schulmaterialien zum neuen Schuljahr.

## Systemabfragen
Mithilfe eines Cronjobs wird nun regelmässig das System nach dessen Befinden befragt. Das Resultat wird in Logfiles gespeichert.
